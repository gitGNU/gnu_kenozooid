Kenozooid — Software stack to utilize capabilities of dive computers

About

Kenozooid is software to plot dive profiles transfered from dive computers. Additional features like performing dive computers backups, changing dive computer configuration or triggering dive simulation are available.

Application developers may find Kenozooid as perfect library for diver's logbook software or other type of applications due its open nature, ability to support multiple dive computers and support for UDDF standard.

Kenozooid is free software licensed under terms of GPL license.

Features

Documentation

User Manual describes how to use Kenozooid to transfer data from dive computers, plot dive profiles or utilize other features of the application.

Software developers interested in using Kenozooid as a library shall study Programmer Manual and API Documentation.

Hacking Guide is written for those interested in Kenozooid internals and development.

Kenozooid uses UDDF format to store data downloaded from dive computers. The UDDF specification may be read at UDDF's website.

Standard Diving Diclaimer

Diving equipment is required to descent under the water for longer periods of time. Correctly used dive computers allow to analyze previous dives and plan the next ones to simplify whole adventure.

Purpose of Kenozooid project is to help with dive planning and understanding diving requirements but it is not and never will be about switching off diver's brain.
